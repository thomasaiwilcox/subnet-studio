# Support

Subnet Studio is a community project rather than a hosted support service.

- For a reproducible defect, open a bug report.
- For suspected arithmetic or aggregation errors, use the CIDR correctness report.
- For a product idea, open a feature request and explain the learner or practitioner problem it solves.
- For security or privacy concerns, follow [SECURITY.md](SECURITY.md) and do not disclose the issue publicly.

When asking for help, include the application version, browser and operating system, workspace CIDR and provider profile, exact steps, and an exported schema-v1 workspace when it is safe to share.

The [live application](https://thomaswilcox.com/subnetvisualiser/) includes searchable help, keyboard shortcuts, guided lessons, scenarios, and Subnet Coach.
