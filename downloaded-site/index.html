Subnet Visualizer — Overview

This tool helps you plan and visualize IPv4 subnets. Set an Address (e.g., 10.0.0.0/16), then split/join blocks. Use Plan to auto‑allocate or View → Next Fit to highlight candidate space for a given /prefix.

  • Address: set the root CIDR envelope
  • Plan: auto‑allocate requests (e.g., “/26 x3”, “hosts=50”)
  • View → Next Fit: highlight where a /prefix can fit
  • Display: labels, animations, and compact mode

What is Next Fit?

Next Fit highlights all leaf blocks that can hold a given size. Pick a /prefix (e.g., /26) and candidates are highlighted. It respects cloud reservation profiles and /31 semantics.

  • Skips allocated blocks
  • Skips leaves where target would have 0 usable IPs under the active profile
  • Use Split on a candidate, or run Plan to place automatically

Keyboard Shortcuts

  • Shift + Drag or Pan Mode — pan the canvas
  • Mousewheel — zoom to cursor
  • Arrow keys — navigate sibling blocks
  • Enter — split selected block
  • Space — multi‑select sibling leaves
